
Objective:
Build a responsive web application that shows current weather and optional 5-day forecast using a public weather API.
Features:
- Input field for city name
- Display of current weather: temperature, humidity, wind, description
- Optional: 5-day weather forecast
- Error handling for invalid input
- Toggleable dark mode for better UI/UX
- Responsive layout (mobile & desktop friendly)

API Used:
OpenWeatherMap API  
Website: https://openweathermap.org/api  
Endpoint examples:
- Current weather: https://api.openweathermap.org/data/2.5/weather
- 5-day forecast: https://api.openweathermap.org/data/2.5/forecast

 How to Use:
1. Replace `YOUR_API_KEY` in `script.js` with your actual OpenWeatherMap API key.
2. Open `index.html` in a browser (preferably with a local server like VS Code Live Server).
3. Type a city name and press "Search".
4. View current weather and 5-day forecast.
5. Toggle  Dark Mode with the button in the top-right.
